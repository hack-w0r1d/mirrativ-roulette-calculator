Mirrativ ルーレットギフト計算機

ルーレットの進行マスの期待値・消費コイン・目標までに必要なギフトの個数の目安を確認できます。

送る個数から計算

送る予定の個数を入力してください。

期待される進行マス数 0.0 マス
消費無償コイン 0
消費有償コイン 0

目標マス数から逆算

期待値ベースで、目標に届く目安の個数を計算します。

必要な個数の目安 0
必要なコイン数の目安 0
期待される進行マス数 0.0 マス
ギフト 必要個数の目安 必要コイン

※ 確率に基づく期待値です。実際の進行マス数は出目によって前後するため、目標達成を保証する数値ではありません。
