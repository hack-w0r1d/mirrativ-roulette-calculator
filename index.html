Mirrativ ルーレットギフト
期待値計算ツール

ルーレットの進行マスの期待値・消費コイン・目標までに必要なギフトの個数の目安を確認できます。

💡SUPER FEVERを使う基準はこちら
期間限定 SUPER FEVER専用 期待値計算

送る個数から計算

送る予定のルーレット個数を入力してください。

期待される進行マス数 0.0 マス
消費無償コイン 0
消費有償コイン 0

使用可能なコインから計算

使用できる無償コイン・有償コインの上限額を入力してください。

期待される進行マス数(目安) 0.0 マス
ギフト 送る個数 消費コイン 期待される進行マス数

※ 有償コインは有償ルーレットに使用する想定です。
※ 有償コインはコインあたりの期待値が高いギフト（有償10連・100連 > 有償単発）から優先して使い切る組み合わせを想定して表示しています。

目標マス数から逆算

期待値ベースで、目標マス数に届く目安のルーレット個数を計算します。

必要な個数の目安 0
必要なコイン数の目安 0
期待される進行マス数 0.0 マス
ギフト 必要個数の目安 必要コイン

※ 確率に基づく期待値です。実際の進行マス数は出目によって前後するため、目標達成を保証する数値ではありません。

計算に使う確率・期待値

無償(100) / 有償(90) 1マス: 40% / 2マス: 45% / 5マス: 10% / 10マス: 5%
期待値: 2.30 マス / 回
有償10連・100連（1連あたり） 2マス: 80% / 5マス: 13% / 10マス: 7%
期待値: 2.95 マス / 回
